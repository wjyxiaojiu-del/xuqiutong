✨ 核心功能

🎯 AI 智能萃取

粘贴客户聊天记录，DeepSeek AI 自动提取结构化需求：痛点、目标、约束、优先级、证据

📄 一键生成 PRD

基于萃取结果自动生成产品需求文档，包含验收标准和 Mermaid 流程图

📊 数据看板

需求统计、客户分布、AI 提效报告可视化展示，一目了然

📁 批量导入

支持 CSV/TXT 文件批量导入客户反馈，飞书文档导入导出

🔄 操作流程

录入客户声音

在「录入需求」页面粘贴客户聊天记录、售前咨询、售后反馈等原始文本，支持选择场景标签

AI 自动萃取

点击「开始萃取」，AI 自动分析文本，提取结构化需求信息，包含痛点、目标、约束、证据等

生成 PRD 文档

在萃取结果页面点击「生成 PRD」，AI 自动生成完整的产品需求文档和流程图

审核与沉淀

支持审核状态流转（待审核→已通过/已驳回）和评论功能，版本管理和回滚已就绪。团队权限、成员管理待接入

🚀 快速开始

# 1. 克隆项目
git clone https://github.com/wjyxiaojiu-del/xuqiutong.git

# 2. 安装依赖
cd xuqiutong && npm install

# 3. 配置环境变量
cp .env.example .env
# 编辑 .env 填入你的 API Key

# 4. 初始化数据库
npx prisma db push

# 5. 启动开发服务器
npm run dev

# 6. 打开浏览器访问
http://localhost:3000
